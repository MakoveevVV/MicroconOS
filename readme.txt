Многозадачность
Пока что реализована кооперативная многозадачность, при которой каждая задача выполняется только после выполнения
предыдущей. В будущем планируется добавить вытесняющую многозадачность а так же реализовать выбор между ними. 

main.c
Используется в данный момент для отладки отдельных частей. В ОС он будет использоваться для составления
работы системы(инициализации периферии, добавление задач в планировщик)
